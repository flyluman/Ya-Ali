## INFO

#### queen.txt:

contains the value of n.

#### n_queen.c:

will input n from queen.txt & output solution & the number of iteration needed for each solution.

#### Compile & Run:

Use [GCC](https://gcc.gnu.org/) to compile n_queen.c

```bash
gcc -Wall -Wextra n_queen.c -o n_queen
./n_queen
```
